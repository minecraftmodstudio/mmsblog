## Sprint 1 ao 3 — Primeira Versão do Modly

Temos o prazer de apresentar o progresso inicial do **Modly**. Com a conclusão dos três primeiros sprints, a base da primeira versão do aplicativo foi estabelecida.

### Sprint 1 — Interface
O primeiro sprint foi dedicado ao desenvolvimento da interface principal do sistema. Foi criada uma estrutura visual organizada e funcional, com o objetivo de proporcionar uma navegação intuitiva.

Também foi iniciado o Editor de Idioma, permitindo suporte futuro a múltiplas línguas.

---

### Sprint 2 — Personalização
O segundo sprint teve como foco a ampliação das opções de personalização. Foram implementadas as seguintes funcionalidades:

- Configuração do ícone do Resource Pack  
- Sistema de configurações do projeto  
- Aplicação de alterações com reinicialização automática  

Esses recursos oferecem maior controle sobre a identidade e configuração dos projetos.

---

### Sprint 3 — Funcionalidades Base
O terceiro sprint concentrou-se na implementação das funcionalidades essenciais do aplicativo. O sistema foi preparado para permitir a criação de Resource Packs de forma visual, sem a necessidade de conhecimento em programação.

Essa etapa representa um avanço significativo em direção ao objetivo principal do projeto: tornar a criação de conteúdo para Minecraft mais acessível.

---

### Versão em Teste
A primeira versão do aplicativo já está disponível para testes. Embora ainda esteja em desenvolvimento, representa um marco importante para o projeto.

---

### Próximos Passos
Nos próximos sprints, estão planejadas as seguintes melhorias:

- Novas funcionalidades de edição  
- Melhorias na interface  
- Suporte a novos tipos de conteúdo  
- Sistema de plugins

## Sprint 1 to 3 — First Version of Modly

We are pleased to present the initial progress of **Modly**. With the completion of the first three sprints, the foundation of the first version of the application has been established.

### Sprint 1 — Interface
The first sprint focused on developing the main interface of the system. A structured and functional visual layout was created to ensure intuitive navigation.

The Language Editor was also initiated, enabling future support for multiple languages.

---

### Sprint 2 — Customization
The second sprint focused on expanding user customization options. The following features were implemented:

- Resource Pack icon configuration  
- Project settings system  
- Application of changes with automatic restart  

These features provide greater control over project identity and configuration.

---

### Sprint 3 — Core Features
The third sprint focused on implementing the core functionalities of the application. The system was prepared to support the creation of Resource Packs through a visual approach, without requiring programming knowledge.

This stage represents a significant step toward the project's main goal: making Minecraft content creation more accessible.

---

### Testing Version
The first version of the application is now available for testing. Although still in development, it represents an important milestone for the project.

---

### Next Steps
In the upcoming sprints, we plan to introduce:

- Additional editing features  
- Interface improvements  
- Support for new types of content  
- Plugin system  

---

Stay tuned for future updates as Modly continues to evolve.
