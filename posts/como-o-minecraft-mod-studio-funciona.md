**O Modly** foi criado para tornar a criação de Resource Packs acessível a qualquer pessoa, mesmo sem experiência com programação ou edição de arquivos JSON.

## Como funciona na prática:


- Abra o app e crie um novo projeto de Resource Pack informando o nome, a versão do Minecraft e o ícone.

- Escolha o que editar — navegue pelas categorias: Blocos, Itens, Mobs, GUI e Ambiente.

- Use o Editor de Pixel Art para texturas 16×16 ou 32×32px com ferramentas de lápis, balde, borracha e conta-gotas.

- Para texturas grandes (GUI, mobs), o Image Editor abre em janela separada com zoom via scroll.

- Edite os textos do jogo com o Editor de Idioma — renomeie itens, blocos e mobs em pt_BR ou en_US.

- Acompanhe tudo no Workspace — cada textura editada aparece como card para fácil re-edição.

- Exporte com um clique — gera um .zip pronto para arrastar para .minecraft/resourcepacks.


`Tudo é mantido em memória durante a sessão. Nenhum arquivo temporário é criado até você exportar. Simples, rápido e sem complicação.`
